Over Queen's Birthday weekend the first Sixty-Four Hour Arcton Coding Extravaganza commenced. Each entrant had 64 hours to code up something to match their themes - The central theme for the weekend was "Choas" and each participant also had a randomly selected second theme they had to incorporate into their work.

Grady Hooker ("Nostalgia")

'Windows Nine-T-8' emulates the Windows 98 user interface within a simulated CRT monitor in a room decorated with game posters and wallpaper straight out of the 1990's. Within the operating system you can read a text file about Chaos and some other gods, listen to some Chaos-themed music, check some emails, launch the classic video game DOOM or trigger a "virus". This virus repeatedly pops up ads from the 1990's on the screen and if you don't close them fast enough you will treated with the classic Windows blue screen of death.
